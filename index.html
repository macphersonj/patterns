The Grid

The basis of the patterns is built around the Bootstrap Grid, found here.   The reason behind using the Bootstrap Grid is to be able to quickly assemble components from this library while being able to customise the design within a logical grid system.     <!-- Content -->
  </div>
  <div class="col-lg-6">
    <!-- Content -->
  </div>
</div>

Notes:

Giving multiple classes to columns will give you control over your mobile views.

For example:

col-lg-6.col-md-6.col-sm-12.col-xs-12

Will yield two columns on large and medium screens (12/6 = 2), and one column stacked on small and extra small screens (12/12 = 1).
